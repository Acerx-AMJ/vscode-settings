## vscode-settings
These are my personal VS Code settings and keybinds.

### Used Extensions
- clangd by LLVM
- cmake by twxs
- Github Theme by Github
- Icons by Mhammed Talhaouy
- rust-analyzer by The Rust Programming Language

### Keybind Cheatsheet
#### Cursor Keybinds
- `ctrl + shift + up` - Add cursor above
- `ctrl + shift + down` - Add cursor down
- `escape` - remove excess cursors
- `ctrl + d` - add cursor to next find match
- `ctrl + shift + d` - add cursor in all matches
- `ctrl + l` - select whole line
- `ctrl + shift + l` - expand selection
- `ctrl + ;` - shrink selection
- `ctrl + u` - cursor undo
- `ctrl + a` - select all

#### Navigation
- `ctrl + shift + .` - navigate file/directory tree
- `ctrl + shift + s` - switch CPP header/source file (clangd)
- `ctrl + o` - open file from files
- `ctrl + k ctrl + o` - open folder
- `ctrl + r` - open recents
- `ctrl + p` - go to file
- `ctrl + shift + p` - run command
- `ctrl + f` - find
- `ctrl + shift + f` - find in files
- `ctrl + h` - replace
- `ctrl + shift + h` - replace in files
- `ctrl + g` - go to line
- `ctrl + j` - go back
- `ctrl + shift + j` - go forward
- `ctrl + shift + \` - go to matching brace
- `ctrl + '` - go to next git change
- `ctrl + shift + '` - go to previous git change
- `ctrl + shift + o` - go to symbol in file
- `ctrl + t` - go to symbol in workspace

#### Line Editing
- `ctrl + shift + k` - delete selected lines
- `ctrl + c` - copy
- `ctrl + x` - cut 
- `ctrl + v` - paste
- `ctrl + ]` - indent line
- `ctrl +[` - de-indent line
- `ctrl + shift + enter` - insert line above
- `ctrl + enter` - insert line below
- `alt + down` - move line down
- `alt + up` - move line up

#### Markdown
- `ctrl + k + ctrl + v` - open preview

#### Other
- `ctrl + k + ctrl + t` - switch color theme
- `ctrl + k + ctrl + s` - edit keybinds
- `ctrl + .` - quick fix
- `ctrl + z` - undo
- `ctrl + shift + z` - redo

#### Comments
- `ctrl + shift + /` - toggle block comment
- `ctrl + /` - toggle single-line comment

#### Editors
- `ctrl + shift + w` - close all editors
- `ctrl + k + ctrl + w` - close all editors in a group
- `ctrl + w` - close editor
- `ctrl + alt + right` - move editor left
- `ctrl + alt + left` - move editor right
- `ctrl + 0` - focus on the sidebar
- `ctrl + 1..9` - focus on the selected group
- `alt + right` - open next editor
- `alt + left` - open previous editor
- `ctrl + shift + tab` - open least recently used editor in group
- `ctrl + tab` - open previous recently used editor in group
- `ctrl + shift + t` - reopen closed editor

#### Sidebar
- `ctrl + b` - toggle sidebar
- `ctrl + shift + g` - show git view
- `ctrl + shift + e` - show explorer
- `ctrl + shift + x` - show extensions
- `ctrl + shift + f` - show search/replace
- `ctrl + \` - split editor
- `ctrl + k + ctrl + m` - toggle editor maximized
